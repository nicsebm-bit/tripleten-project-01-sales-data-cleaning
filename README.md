# 📊 TripleTen Project 01

# Sales Data Cleaning & Business Analysis

## 📌 Project Overview

This project was completed as part of the TripleTen Data Analytics Bootcamp.

The objective was to transform raw e-commerce sales data into a clean and reliable dataset ready for analysis. The project includes data cleaning, KPI calculation, business insights and data visualizations using Google Sheets.

---

## 🎯 Business Problem

VentaExpress, an e-commerce company operating in Mexico and Colombia, needed to prepare an executive report for the last quarter of 2024.

The available dataset contained inconsistent formatting, duplicate records, missing values and unstructured information that required cleaning before any meaningful analysis could be performed.

## 🛠️ Tools Used

- Google Sheets
- Data Cleaning
- Data Validation
- Data Analysis
- Data Visualization
- Business Reporting

---

## 📂 Dataset

**Source:** TripleTen Bootcamp

**Period:** October – December 2024 (Q4 2024)

**Industry:** E-commerce

**Countries:** Mexico and Colombia

The dataset contains sales transactions including product information, cities, quantities, unit prices and total sales.

---

## 🧹 Data Cleaning Process

To ensure data quality and reliability, the following cleaning tasks were performed:

- Removed duplicate records.
- Standardized city names using proper capitalization.
- Split product descriptions into Category, Type and Specifications.
- Identified and completed missing values in **Unit Price** and **Total Amount**.
- Applied consistent currency formatting.
- Organized the workbook into dedicated sheets for raw data, cleaned data, analysis, visualizations and executive report.

These steps ensured that the dataset was accurate, consistent and ready for analysis.

---

## 📈 Business Analysis

After cleaning the dataset, several business metrics were calculated, including:

- Total quarterly sales.
- Average revenue per transaction.
- Total number of transactions.
- Best-selling product category.
- City with the highest sales.
- Best-performing month.
- Average product price by category.

These metrics provided a clear overview of the company's sales performance during the fourth quarter of 2024.
